# nbutapp
 on test [beta]1.0.0

## 项目简介
本平台是由*summerpray*提供的nbut外卖快递悬赏服务小程序全栈项目，每个人都可以在上面发布订单，但是由于作者承担不起服务器的费用和无法作为商业用途，暂时没有发布小程序。

### 功能一览：
1. 创建订单
2. 发布订单
3. 接取订单
4. 显示历史订单
5. 删除发布过的所有订单
6. 删除指定订单
7. 订单详情
8. 反馈功能
9. 修改订单（暂时未嵌入）
10. 帮助（暂时未嵌入）
11. 钱包功能（暂时未嵌入）
12. *支付功能（暂时未嵌入）*
